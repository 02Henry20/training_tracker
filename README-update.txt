Ascend update 0.4.1

Changes:
- Settings save automatically; save button removed.
- Header mark now shows current rank instead of AR.
- Desktop header hides the redundant rank text.
- Guided tutorial button added in Setup. Tutorial uses tutorial-data.json only in memory and restores user data after exit.
- Manifest now lists Android circular icons first and includes transparent favicons for browser tabs.

Note: Android launchers may cache PWA icons. After replacing icons, uninstall the old installed PWA/shortcut and install it again if Android still shows the old square icon.
